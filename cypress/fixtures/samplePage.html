Complex Operations w/ Defaults

Here are two examples of how to create very complex operations, multiple watermarks and blends using multiple imgix sources and hidden default parameters.

The "g" watermarks are defaults of the exampletext.imgix.net sources, as well as the text formatting. Since these are constant, only the variable aspects of the images get exposed in the URL. Click the button below each image to view the parameter breakdown using imgix sandbox.
